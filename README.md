# 🏥 NIDUS EXIM Pvt. Ltd. — Corporate Website

A professional corporate website built for a medical infrastructure 
company specializing in Operating Theatre construction and design.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL (PDO)

## ✨ Features
- Multi-page responsive website (8+ pages)
- Contact form with server-side validation
- SQL injection prevention using PDO prepared statements
- Admin panel to view customer enquiries
- Mobile-friendly navigation with hamburger menu
- Scroll animations using Intersection Observer API
- Reusable header/footer component system

## ⚙️ Setup Instructions
1. Clone the repository
   git clone https://github.com/yourusername/nidus-exim-website.git

2. Import the database
   - Open phpMyAdmin
   - Import `nidus_exim_db.sql`

3. Configure database in `db.php`
   - Set your DB_HOST, DB_USER, DB_PASS

4. Run on localhost using XAMPP or WAMP
